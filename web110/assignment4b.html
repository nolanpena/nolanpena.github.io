I noticed that using long passwords, usually between 16 and 20 characters, is more beneficial than focusing simply on complexity. Furthermore, enabling multi-factor authentication increases security by requiring a second verification step, such as a number texted to a phone. Most accounts I have already have multi factor authentication on them, though my passwords are not very strong. I plan to implement these security measures on my Apple, Facebook, Google, and Microsoft accounts, as for the password manager I will probably just use the one built into my phone. Overall, these practices will significantly enhance my online security and protect my sensitive information from potential threats.
