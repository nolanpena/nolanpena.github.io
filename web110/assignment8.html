Assignment 8: SQL Statements

Summary

SQL SELECT Statement

This SQL statement retrieves all records from the CUSTOMERS table.

  • SELECT * FROM CUSTOMERS;

SQL WHERE Clause

This SQL statement retrieves records from the CUSTOMERS table where the Country is 'USA'.

  • SELECT * FROM CUSTOMERS WHERE Country = 'USA';

SQL COUNT Function

This SQL statement counts the number of customers in the CUSTOMERS table.

  • SELECT COUNT(*) FROM CUSTOMERS;
SQL INSERT INTO Statement

This SQL statement inserts a new record into the CUSTOMERS table.

  • INSERT INTO CUSTOMERS (CustomerName, Country) VALUES ('New Customer', 'USA');

SQL UPDATE Statement

This SQL statement updates a specific record in the CUSTOMERS table.

  • UPDATE CUSTOMERS SET Country = 'Canada' WHERE CustomerName = 'Some Customer';
